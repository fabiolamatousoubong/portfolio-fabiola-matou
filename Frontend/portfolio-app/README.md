# Portfolio Fabiola

Dieses Projekt ist ein Portfolio, das mit Nuxt 3 und einem Java-Backend entwickelt wurde.

## 🚀 Technologien
- Vue / Nuxt 3
- Java Spring Boot
- Docker

## 📂 Funktionen
- Online-Lebenslauf
- Upload von Dokumenten
- Persönliches Profil

## 🌐 Zugriff
Projekt befindet sich derzeit in Entwicklung (localhost)
